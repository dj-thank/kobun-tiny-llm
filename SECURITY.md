# Security

Do not publish secrets, API keys, raw licensed corpus exports, or local
checkpoints with optimizer state.

Before publishing:

```powershell
rg -n "(api[_-]?key|secret|token|password|bearer|BEGIN .*PRIVATE)" -g "!release/**" -g "!.venv/**" .
```

For model release folders, publish only inference artifacts generated by
`scripts/export_hf_release.py`.
